When a reagent item arrives, a stock can be created. Alternatively, when a new batch of in-house reagent is made, a stock can be created. Stock is a 'container' for the reagent item or in-house reagent.

  • Start typing in the item or In-house reagent box to search for an existing reagent. If the reagent does not exist, click on the button to open a new window and create a new reagent.
  • Enter the lot number. For in-house reagents, this is often the date when the reagent is made in ddmmyyyy format.
  • Enter the delivery date. For in-house reagents, this is the date when you commit it to the stocks.
  • Select the condition of the stock.
  • Add any necessary comments. For example, if the condition of the stock is not Good, you can write a brief description here.
  • Enter an expiry date according to the commercial reagents or the SOP for in-house reagents.
  • Enter the quantity received/created.
  • Type in the location box to enter where the stock will be stored. If the stock needs to be stored in multiple locations, enter one location first, save, and then edit each individual stock entry to add the other locations.
  • If a location does not exist, click on the button to open a new window and create a new entry.
  • Click to save and continue.
  • Correct any errors that occur and save again.
