To update an existing item, modify the form fields with the necessary information.

  • Update the item name in the Name field.
  • Update the product ID in the Product ID field.
  • Update the CAS number in the CAS number field, if applicable. CAS numbers are unique identifiers for chemical compounds. They help distinguish between different compounds with very similar names.
  • Update the quantity and select the appropriate unit in the Quantity and Quantity Unit fields.
  • To update the category, delete the current value and start typing in the Category search box to find an existing category. If the category does not exist, click on the button to create a new category.
  • To update the manufacturer, delete the current value and start typing in the Manufacturer search box to find an existing manufacturer. If the manufacturer does not exist, click on the button to create a new manufacturer.
  • To update the supplier, delete the current value and start typing in the Supplier search box to find an existing supplier. If the supplier does not exist, click on the button to create a new supplier.
  • Update the description in the Description field to provide additional details about the item.
  • Click the button to save the updated item.
  • Correct any errors that occur and save again.
