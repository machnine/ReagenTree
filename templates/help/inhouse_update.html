To update an existing in-house reagent, modify the form fields with the necessary information.

  • Update the reagent name in the Name field, delete the current reagent and start typing to search a new one.
  • Update the product ID in the Product ID field.
  • Update the lot number in the Lot Number field.
  • Update the expiry date in the Expiry Date field.
  • Update the quantity and select the appropriate unit in the Quantity and Quantity Unit fields.
  • To update the category, start typing in the Category search box to find an existing category. If the category does not exist, click on the button to create a new category.
  • Update the description in the Description field to provide additional details about the reagent.
  • Click the button to save the updated in-house reagent.
  • Correct any errors that occur and save again.
