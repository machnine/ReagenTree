To create a new in-house reagent, fill out the form fields with the required information.

  • Enter the reagent name in the Name field. start typing in the Name box to find an existing reagent.
  • Enter the product ID in the Product ID field.
  • Enter the lot number, usually the creation date in ddmmyyyy format, in the Lot Number field.
  • Enter the expiry date according to the relevant SOP in the Expiry Date field.
  • Enter the quantity and select the appropriate unit in the Quantity and Quantity Unit fields.
  • To assign a category, start typing in the Category box to find an existing category. If the category does not exist, click on the button to create a new category.
  • Enter a description in the Description field to provide additional details about the reagent.
  • Click the button to save the new in-house reagent.
  • Correct any errors that occur and save again.
