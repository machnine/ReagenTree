To create a new item, fill out the form fields with the required information.

  • Enter the item name in the Name field.
  • Enter the product ID in the Product ID field.
  • Enter the CAS number in the CAS number field, if applicable. CAS numbers are unique identifiers for chemical compounds. They help distinguish between different compounds with very similar names.
  • Enter the quantity and select the appropriate unit in the Quantity and Quantity Unit fields.
  • To assign a category, start typing in the Category search box to find an existing category. If the category does not exist, click on the button to create a new category.
  • To assign a manufacturer, start typing in the Manufacturer search box to find an existing manufacturer. If the manufacturer does not exist, click on the button to create a new manufacturer.
  • To assign a supplier, start typing in the Supplier search box to find an existing supplier. If the supplier does not exist, click on the button to create a new supplier.
  • Enter a description in the Description field to provide additional details about the item.
  • Click the button to save the new item.
  • Correct any errors that occur and save again.
