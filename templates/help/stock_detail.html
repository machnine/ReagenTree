Stock Details

Click the icon to set a stock level warning threshold.

Hover over the icon to view the set threshold value. Click the to confirm the deletion of the set threshold.

A warning threshold can only be set or deleted, so if a different threshold is needed instead of the one currently set. Delete it first and then create a new one.

Click the button to edit stock details.

Please note: the source of the stock is not editable. If a stock source needs to be changed, create a new stock item and delete the incorrect one.

Validations

To add a validation record for the stock, click the button.

Any user can add, edit, or remove a validation record. Click the icon to display the and buttons.

Only administrators can authorise a validation record. To authorise, click the icon and confirm the action.

Please note: only validated stock items or those marked as validation not required can be used to create a reagent.
